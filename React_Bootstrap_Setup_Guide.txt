2. Creating the Project
------------------------------------------------

You have two options. Vite is recommended since itâ€™s faster and more modern.

Option A: Vite (Recommended)
----------------------------
# 1. Scaffold the project
npm create vite@latest random-dictionary-app

# Choose options:
# - Framework: React
# - Variant: JavaScript

# 2. Move into folder
cd random-dictionary-app

# 3. Install dependencies
npm install

Option B: Create React App (CRA)
--------------------------------
npx create-react-app random-dictionary-app
cd random-dictionary-app

(CRA still works, but itâ€™s slower and less flexible than Vite.)

------------------------------------------------
3. Add Bootstrap
------------------------------------------------

Vite projects
-------------
npm install bootstrap

In src/main.jsx, import Bootstrap:
import 'bootstrap/dist/css/bootstrap.min.css';

CRA projects
------------
npm install bootstrap

In src/index.js, import Bootstrap:
import 'bootstrap/dist/css/bootstrap.min.css';

------------------------------------------------
4. Run the Project
------------------------------------------------

Once everything is installed:

Vite
----
npm run dev

Vite will print a local server URL, usually:
http://localhost:5173

CRA
---
npm start

Runs on:
http://localhost:3000
